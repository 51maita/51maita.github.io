五分钟拥有自己的免费博客

2020年3月13日
| 教程
| 阅读

1. 前言

      意叔我作为多年老码农，一直想拥有属于自己的博客。但每每想着要搭建一个博客要购买服务器（租用虚拟空间），要搭建博客网站，要备案，还要购买域名…就直接打退堂鼓了。直到有一天遇到了Hugo 和 Github Pages。

Hugo: 一个开源的静态网站生成器，号称是世界上最快的创建网站的框架。其基于go语言开发，能够超快的生成静态页面。

Github Pages: Github是全部最大的开源网站，别称“世界上最大的同性交友网站”。上面有无数优秀的开源项目，是码农的最大宝库！而 Github Pages则是Github提供的一个创建免费博客的站点。

      本文记录了意叔如何利用了Hugo 和 Github Pages搭建了自己的免费博客。供想快速搭建一个属于自己的免费博客的同学参考。

2. 软件准备

本文基于 windows (win7 64位) 操作系统。如果你是32位的操作系统，则下载对应的软件包即可。

2.1 下载和安装Git客户端

  1. Git客户端(版本:2.25.1): 64位下载地址   32位下载地址
  2. 双击安装(譬如安装到d:\Program Files\Git;)
  3. 将git命令路径(d:\Program Files\Git\cmd)添加到系统环境变量Path中

2.2 下载和安装Hugo

  1. Hugo(版本:0.67.0)： 64位下载地址  32位下载地址
  2. 解压到指定目录(譬如：E:\blog,==本文后续的操作都基于本目录==)
  3. 将解压路径(E:\blog)添加到系统环境变量Path中

3. 创建网站

3.1 生成站点

  1. 进入命令控制台窗口(开始->运行-> 输入 ‘cmd’ 回车 )

推荐使用cmder 代替系统默认的命令行窗口

  1. 进入要创建网站的目录

cd E:\blog

  1. 生成站点(以下命令生成名为【51maita】的站点)

hugo new site 51maita

  1. 进入站点目录

cd 51maita

3.2 安装皮肤

可以到hugo官网的 皮肤列表 挑选一个心仪的皮肤，找到对应皮肤的 GitHub地址，创建目录themes，在themes目录里把皮肤 git clone 下来即可。 以下命令安装 ==even== 皮肤,其地址为：https://github.com/olOwOlo/hugo-theme-even

  • cd themes ==进入主题皮肤目录==
  • git clone https://github.com/olOwOlo/hugo-theme-even ==等待下载完成后即可配置皮肤==

3.3 配置皮肤

将 themes/==even==/exampleSite目录下的config.toml复制到网站根目录(blog/51maita)(==存在则覆盖==),并根据自身情况修改对应的配置即可。

  • copy themes\even\exampleSite\config.toml .\config.toml

修改config.toml文件，设置 theme===even==

3.4 创建文章

hugo new post/first.md ==在post目录创建first.md的文章==

3.5 本地运行

hugo server –theme=even –buildDrafts ==成功的话，即可通过http://localhost:1313访问==

4. 发布网站

      网站最终是要让他人可以访问的，只在本地可以浏览意义不大，接下来我们看如何把网站发布到github pages，让全世界的人都可以访问。

4.1 创建github仓库

  1. 我们需要到Github上申请一个账号。譬如意叔申请了一个用户名为==51mailta==的账号。
  2. 创建一个名为：用户名.github.io 的仓库，如意叔的用户名为51maita，则创建一个 51maita.github.io 的仓库(须是公开的参控股,即public模式）

4.2 生成网站

hugo –theme=even –baseUrl="https://51maita.github.io” –buildDrafts

参数说明

  • –theme 指定主题
  • –baseUrl 指定网站首页地址
  • –buildDrafts 创建草稿(可选)

执行以上命令后会在网站目录生成一个public文件夹，里面的内容即是我们的博客网站文件，接下来我们将其发布到Github Pages。

4.3 发布网站

  1. 进入public目录

cd public

  1. 初始化本地仓库

git init

  1. 将所有文件添加到版本控制

git add .

  1. 提交到本地仓库

git commit -m “初始化提交”

  1. 将本地仓库和github 网站关联起来

git remote add origin https://github.com/51maita/51maita.github.io

  1. 发布到github网站

git push https://github.com/51maita/51maita.github.io master

  1. 发布成功

浏览器输入 https://51maita.github.io 即可访问自己的博客了。

5. 个性化

5.1 自定义域名

  1. 申请一个自己的域名(可以到阿里云或腾讯云申请),譬如 www.51maita.com
  2. 登录Github，选择并进入[用户名].github.io的仓库。如：51maita.github.io
  3. 点击 [Settings] 进入设置,
  4. 拖动滚动条到 [GitHub Pages] 设置 [Custom domain] 为自己的域名如: www.51maita.com
  5. 点击保存即可通过自己的域名访问博客了。

5.2 支持HTTPS

  1. 登录域名管理后台，并将github pages的域名的A记录的值修改为下面IP的其中一个:
  • 185.199.108.153
  • 185.199.109.153
  • 185.199.110.153
  • 185.199.111.153

==如果原来用的不是A记录，则删除新增。==

  1. 重新配置github pages的域名
  • 进入github pages的仓库的settings页面
  • 找到 GitHub Pages 的配制区域,删除之前配制的域名,保存.
  • 设置同样的域名,再保存.
  • 此时 【Enforce HTTPS 】选项是不可勾选状态,并且提示–Not yet available for your site because the certificate has not finished being issued (自定义域名) .
  • ==等大概两个多小时….==
  • 此时【Enforce HTTPS 】选项应该可以勾选了。
  • 勾选上则可以使用https访问自定义域名了。

5.3 皮肤与插件

  1. HUGO 提供了很多皮肤，可以到官网选择自己中意的皮肤。并可以根据自己的需要进行个性化配置和修改，具体请参考HUGO官方文档，限于篇幅在此就不多做说明了。
